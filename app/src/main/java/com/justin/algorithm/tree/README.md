# 树

## 二叉树

需要了解的点：
* 递归非递归先序、中序、后序遍历
* 宽度优先遍历
* 求最大宽度


### 二叉搜索树

* 若它的左子树不为空，则左子树上所有节点的值都小于根节点的值
* 若它的右子树不为空，则右子树上所有节点的值都大于根节点的值
* 它的左右子树也分别为二叉搜索树

中序遍历是升序

### 完全二叉树

* 任何一个节点如果有左节点，则必有右节点
* 当采用广度优先遍历时，遇到了第一个有左无右的节点，则后序所有的节点都是叶子节点

### 满二叉树
* 除了叶子节点，所有的节点都是有左有右
* 总节点数 n = 2^(树的深度) - 1

### 平衡二叉树（AVL树）
* 左子树和右子树的高度之差的绝对值小于等于1
* 左子树和右子树也是平衡二叉树

### 树形DP 
DP：动态规划
采用**递归**解决
