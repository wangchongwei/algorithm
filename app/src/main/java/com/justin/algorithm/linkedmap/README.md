# 链表算法题

## 链表简介

### 单链表
每个节点有一个指针
* next指向下一个节点

### 双向链表
每个节点有两个指针
* next指向下一个节点
* prev指向上一个节点

### 环形链表
如果链表存在环形，进入环形则无法跳出，环形肯定在链表的末端
                1
                |
                2
                |
                3
               /  \
              4    6
               \  /
                5

## 常见解题思路
双指针
快慢指针
